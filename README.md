# 🧢 Pokedex App

Simple Pokedex application built using HTML, CSS and JavaScript.

## 🚀 Live Demo
https://pawelkobialka.github.io/pokedex-app/

## 🔍 Features
- Search Pokemon by name
- Display Pokemon image
- Show Pokemon type
- Dynamic background color based on type
- Suggestions while typing
- Link to Cardmarket

## 🛠 Technologies
- HTML
- CSS
- JavaScript
- PokeAPI
